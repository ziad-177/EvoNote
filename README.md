
# EvoNote 🧠📝

EvoNote is an AI-powered smart note-taking application designed to help users summarize, refine, and organize their ideas effortlessly.

#  Architecture (Microservices)
This project is built using a modern microservices architecture, containerized with Docker for easy deployment and scaling.

## Getting Started
 First: Make sure you have prepared all the files and configured Docker Compose correctly 

 Second: Open the EvoNote-Main folder

 Finally: Run the code using docker-compose up
